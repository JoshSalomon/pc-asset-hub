## Deployment from Claude Code sandbox

Claude Code runs in a container without podman/docker. To build and deploy:

```bash
ssh -i ~/.ssh/container-dev -o StrictHostKeyChecking=no jsalomon@host.containers.internal deploy
```

This runs `kind-deploy.sh rebuild "kubectl --context kind-assethub"` on the host with podman available.

## Live tests after deploy

kubectl works directly from the sandbox, but the API/UI are on the host network:
- API: `http://host.containers.internal:30080`
- UI: `http://host.containers.internal:30000`

Run live tests with the host URL:
```bash
bash scripts/test-publishing.sh http://host.containers.internal:30080
```

## Note
`localhost` does NOT work from the sandbox — always use `host.containers.internal`.
